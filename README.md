# car-record

Pure iOS client skeleton for a local-first car management app.

## Scope

- SwiftUI app architecture
- Local persistence with SwiftData
- No backend dependency
- Offline-first data flow

## Directory

- `ios/CarRecord/App`: app entry and root tabs
- `ios/CarRecord/Models`: SwiftData models
- `ios/CarRecord/Persistence`: container and persistence setup
- `ios/CarRecord/Features`: feature-level UI
- `ios/CarRecord/Shared`: shared helpers

## Next Step

1. Open Xcode and create an iOS App shell project named `CarRecord`.
2. Add files under `ios/CarRecord` into the Xcode target.
3. Set deployment target to iOS 17.0+ (SwiftData).
4. Build and run on simulator/device.
